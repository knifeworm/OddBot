# OddBot
OddBot is a discord.py bot written in python

## Installation
1. Download the zip files.
2. Run this command:
```bash
npm i 
```
3. Add all the required information into config_example.json
4. Rename your config_example.json file to config.json
5. Run this command:
```bash
node bot.js
```
And your all set.
